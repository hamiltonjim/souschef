NumberReader

object NumberReader

Object that reads a number, consisting of (potentially) a whole part and a fraction part, either of which may be missing. This object will recognize all the following as "three and a half:"

  1. 3.5

  2. 3 1/2

  3. 3½

  4. 3⅟2

  5. 3 ⅟2

  6. 3¹⁄₂

  7. 3 ¹⁄₂

In the second form, both the ASCII / (0x27) and the Unicode FRACTION_SLASH will be recognized as separating numerator and denominator. In the fourth form, the Unicode FRACTION_NUMERATOR_ONE starts the fractional part, without a space; the fifth form adds a space. Versions 6 and 7 use the SUPER_ONE and SUB_TWO integers, with the FRACTION_SLASH; without and with a space character, respectively.

Note: any "normal", superscript, or subscript integer will "work" in any position, without regard to whether it's in the numerator, denominator, or integer part.

The names of the const properties are obvious, so not individually commented.
